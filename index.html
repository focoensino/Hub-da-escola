Como usar este portal

  1. Procure a ferramenta que você precisa usando a barra de pesquisa
  2. Clique no card da ferramenta para abri-la em uma nova aba
  3. Para voltar ao portal, basta fechar a aba da ferramenta
  4. Use o portal sempre que precisar acessar rapidamente as ferramentas das aulas
